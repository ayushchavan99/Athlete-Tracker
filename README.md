# Athlete Tracker

A C-based athlete tracking system designed for athletes and kabaddi players.

## Features

### V1

* Add Daily Entry
* View All Entries
* Show Best Sprint
* Show Discipline Score
* Show Weight Progress

### V2

* Save Entries to File
* Load Previous Entries Automatically
* Input Validation
* Persistent Data Storage

### V3

* Automatic Date Tracking
* Automatic Day Tracking
* Entry History with Date and Day

## Data Tracked

* Date
* Day
* Weight (kg)
* Protein Goal Completion
* Workout Completion
* Practice Completion
* Sprint Time

## Technologies Used

* C Programming
* Structures
* Arrays
* Functions
* Loops
* File Handling
* Time Library (`time.h`)
